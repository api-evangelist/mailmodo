# Mailmodo (mailmodo)

API and integration profile for **Mailmodo** — an AI-powered interactive email marketing and automation platform headquartered in Bengaluru with a presence in San Francisco. Mailmodo pioneered AMP-for-Email at scale, letting brands embed forms, quizzes, polls, carousels, and calendars directly inside the inbox.

## APIs

### Mailmodo REST API

- Base URL: `https://api.mailmodo.com`
- Authentication: API key passed in the `mmApiKey` request header
- Operations cover Contacts, Contact Lists, Campaigns, Templates, and custom Events
- Full OpenAPI spec: [openapi/mailmodo-openapi.yml](./openapi/mailmodo-openapi.yml)
